Transaction

By default, this is a String containing the identifying hash of the transaction.
If transactions are expanded in binary format, this is an object whose only field is tx_blob, containing the binary form of the transaction as a decimal string.
If transactions are expanded in JSON format, this is an object containing the transaction object including the transaction's identifying hash in the hash field.
